# NBA2K Reputation Leveling Service

## 🎯 What is Reputation Leveling?

Reputation leveling is a core progression system in NBA2K that unlocks exclusive rewards, animations, and badges as you advance through different tiers.

## 📊 Level Tiers

### 🏀 Rookie Tier
- **Rookie 1 → Rookie 2**: 100 reputation
- **Rookie 2 → Rookie 3**: 150 reputation
- **Rookie 3 → Rookie 4**: 250 reputation (Unlock: Remove Shirt)
- **Rookie 4 → Rookie 5**: 300 reputation (Unlock: +1 Badge Perk)
- **Rookie 5 → Starter 1**: 1,000 reputation

### ⭐ Starter Tier
- **Starter 1 → Starter 2**: 1,200 reputation
- **Starter 2 → Starter 3**: 1,500 reputation (Unlock: +1 Badge Perk)
- **Starter 3 → Starter 4**: 1,800 reputation (Unlock: Animations)
- **Starter 4 → Starter 5**: 2,200 reputation (Unlock: +3 Badge Perks)
- **Starter 5 → Veteran 1**: 3,000 reputation

### 🎖️ Veteran Tier
- **Veteran 1 → Veteran 2**: 3,500 reputation
- **Veteran 2 → Veteran 3**: 4,000 reputation (Unlock: +5 Badge Perks)
- **Veteran 3 → All-Star 1**: 5,000 reputation (Unlock: Exclusive Animations)

### 🌟 All-Star Tier
- **All-Star 1 → All-Star 2**: 6,000 reputation
- **All-Star 2 → All-Star 3**: 7,000 reputation (Unlock: All-Star Badge)
- **All-Star 3 → Superstar 1**: 8,000 reputation (Unlock: Exclusive Gear)

### 👑 Superstar Tier
- **Superstar 1 → Superstar 2**: 10,000 reputation
- **Superstar 2 → Superstar 3**: 12,000 reputation (Unlock: Superstar Badge)
- **Superstar 3 → Legend**: 15,000 reputation (Unlock: All Unlocks)

### 🏆 Legend

## 🎮 How to Calculate Price

Use the Discord bot command:
```
!reputation <current_level> <current_percent> <target_level> <target_percent> [platform]
```

### Example:
```
!reputation "Rookie 1" -4 "Starter 1" -3 PC
```
This calculates: From Rookie 1 (need to fill 4%) to Starter 1 (stop at 97%) on PC platform.

### Understanding Percentages:
- **-4** = You need to fill 4% of current level
- **0** = Level is complete
- **-3** = Stop at 97% of target level
